Instructions

Proceed in the following order!

  1. Set number of arrows.
  2. Set difficulty level.
  3. Press START button.
  4. Use arrow keys to catch the moving arrows!
  5. Before starting over, press RESTART!

Demo - Catch the Arrow

This minigame was inspired by the "Life's a beach" mission from GTA San Andreas, where the player has to push correct arrows in time to win a dancing competition. I call it "Catch the Arrow"

Catch the Arrow is implemented on GTA V Role Play private server and applied to various activities (such as repairing a car or resuscitating other players) that the player would normaly randomly pass or fail. Instead, you have to win the minigame to succeed.
